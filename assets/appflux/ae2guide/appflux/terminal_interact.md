# Взаємодії у терміналі

Термінали відображатимуть накопичену енергію у вигляді цієї _зеленої рідини_.

![FE_GUI](../pic/fe_terminal.png)

Це не справжня рідина, і ви не можете використовувати відра для її заповнення.

Ви можете використовувати енерговмісний предмет, такий як батареї, електричні інструменти, для заповнення/зливання їхньої енергії.

![FE_GUI2](../pic/fe_terminal2.png)
