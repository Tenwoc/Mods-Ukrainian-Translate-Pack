# Кульки матерії

<ItemImage id="matter_ball" scale="4" />

Кулька звичайної матерії, корисна як боєприпас для [матеріальної пушки](matter_cannon.md) або для виробництва [фарбових кульок](paintballs.md).

Створюється з 256 предметів чи відер в [конденсаторі матерії](condenser.md) у режимі кульок.
