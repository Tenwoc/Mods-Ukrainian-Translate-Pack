# Бездротова точка доступу

<BlockImage id="wireless_access_point" p:state="has_channel" scale="8" />

Дозволяє бездротовий доступ через <ItemLink id="wireless_terminal" />.
Дальність дії, а з нею й споживання енергії можна збільшити, додаючи в інтерфейс цього пристрою <ItemLink id="wireless_booster" />.

Мережа може мати будь-яку кількість бездротових точок доступу з будь-якою кількістю бездротових підсилювачів у кожній з них, що дозволяє оптимізувати споживання енергії та дальність дії через перерозподілення установки.

Цьому пристрою потрібен [канал](../ae2-mechanics/channels.md).

Також використовується для прив'язки [бездротових терміналів](wireless_terminals.md)

# Бездротовий підсилювач

<ItemImage id="wireless_booster" scale="2" />

Використовується для збільшення дальності бездротової дії точки доступу.
